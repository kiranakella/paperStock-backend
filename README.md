# paperStock-backend

Spring Boot 3 + Java 21 backend for the Paper Trading application.

## Stack
- GraphQL
- MongoDB
- Redis caching
- JWT security

## Run
- Start MongoDB on `mongodb://localhost:27017`
- Start Redis on `localhost:6379`
- Run the app with Maven
